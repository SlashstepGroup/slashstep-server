# Slashstep Server security policy
## Supported versions
Since Slashstep Server is still in its initial development stage, the [main branch](https://github.com/SlashstepGroup/slashstep-server) is the only supported version of Slashstep Server.

## Reporting a vulnerability
To securely report a vulnerability, please send an email to christian.toney@beastslash.com.

## Suggesting non-urgent security enhancements
For general security enhancement suggestions that don't need immediate attention, please [create an issue](https://github.com/SlashstepGroup/slashstep-server/issues/new).
