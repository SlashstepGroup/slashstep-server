# Installing Slashstep Server with a binary
> [!WARNING]
> Slashstep Group recommends containerizing Slashstep Server to protect your computer. If you plan to use Slashstep Server in a production environment, consider installing it with [Podman](./installing-slashstep-server-with-podman.md) instead.

## Prerequisites
* A Windows or Linux computer with access to the Internet
* A server running [PostgreSQL 18](https://www.postgresql.org/download/)

## Steps
### Step 1: Download the binary
You can download the binary from the [releases page](https://github.com/SlashstepGroup/slashstep_server/releases) on the GitHub repository.

### Step 3: Set up Slashstep Server through the console
